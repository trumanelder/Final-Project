# Final Project - C++ Programming

## Overview
This is a group project implementing two C++ programming assignments from Big C++: Late Objects.

## Projects

### P8.2 - Spell Checker
A program that checks the spelling of all words in a file against a dictionary word list.

**Features:**
- Reads a dictionary file into memory
- Checks each word in an input file against the dictionary
- Prints out all words not found in the dictionary

### P8.7 - Monoalphabet Cipher
A program that encrypts or decrypts files using a keyword-based monoalphabet cipher.

**Features:**
- Encrypts and decrypts files using a keyword
- Command-line interface: `crypt -d -kKEYWORD input.txt output.txt`
- More secure than simple Caesar cipher
